## 6. 规则 & SOP

### 内容生产
- **分级**: A20%(2-3h)/B50%(1-1.5h)/C30%(30min)
- **EEAT SOP**: 作者署名 + 更新日期 + 数据来源 + 联盟披露(每篇必带)
- **措辞**: 禁止 tested/hands-on → 统一 research-backed/analyzed
- **发布后**: GSC索引请求 + Pin排期 + 内链检查 + GEO citation cue

### SEO
- **选品**: Amazon+Google Trends双验证 + BSR<50K + 评分≥3.8 + Trends>3
- **KD策略**: <20优先 / 20-30谨慎 / >30等DA到15+
- **Sitemap**: trailingSlash:"never", 98 URL收录

### GEO
- **AI爬虫**: GPTBot/ChatGPT-User/PerplexityBot/Google-Extended/CCBot/anthropic-ai/*
- **Schema**: 5种类型全站覆盖(Organization/Review/Product/Article/FAQPage/BreadcrumbList)
- **AI引用因子**: 问题式H2(+2.8×) / dfn定义(+57%) / 对比内容(+55%) / 统计数据(+62%)
- **llms.txt**: 6/18新建，含全站页面索引 + llms-full.txt扩展摘要
- **月度复检**: geo-audit重打分 + llms-full.txt更新 + Schema抽查 + 品牌实体进度

### 社交
- **Pinterest**: P1, 每天 1-2 Pin 分散发布(不集中发), 每周8-12 Pin, UTM必须, Board SEO优先, Alt text+123%点击, 220-232字符描述最优。周六批量做图, 周一到周日分散发
- **FB Groups**: P2, 5min/天, 8群浏览/点赞/纯帮助, D15+可提及
- **Reddit**: P2, 5min/天, D14解禁可放链接, 90/10法则, 绝不自动化
- **Medium**: 暂停(3篇canonical够)

### 外链
- **每周固定1-2h**: Week1目录→Week2竞品分析→Week3资源页→Week4 Guest Post调研
- **持续**: HARO/Qwoted, 每周2-3条回复
- **目标**: 月+10-20外链

### 技术
- **部署**: Vercel(`muchengxian-labs-projects/smartpetguide`), GitHub push自动部署
- **URL**: trailingSlash:"never", canonical去斜杠
- **PageSpeed**: 手机86/桌面99

### 项目管理
- **文件同步**: task_plan/progress/findings/dashboard/project memory 每次改动必更新
- **每日**(~2h): 养号15min + 内容1-1.5h + Pin 10min
- **每周**(~5h): NR扫描×2 + 外链1-2h + Pin分析 + GSC×2 + GEO存活检查
- **每月**(~2.5h): GSC深度 + Pin月度 + Semrush KD + 选题池 + 外链回顾 + GEO复检

## 7. 踩坑经验

### 技术错误

| 错误 | 修复 | 教训 |
|------|------|------|
| vercel.json路由全站404 | cleanUrls:true+trailingSlash:false | 先理解构建输出再配路由 |
| 24个假ASIN全部无效 | 浏览器逐个验证真实ASIN | 绝不编造数据 |
| Vercel vs Netlify记错平台 | 读响应头确认→连接GitHub | 部署前验证，别凭记忆 |
| GSC 4重定向错误 | trailingSlash:"never" | Day 10-14必做：尾部斜杠统一 |
| 8内部坏链(92处引用) | 统一修正所有slug | 定期审计内部链接 |
| WebFetch预检被Cloudflare拦截 | skipWebFetchPreflight:true+VPN TUN | GEO审计需特殊配置 |

### 策略错误

| 错误 | 修正 | 来源 |
|------|------|------|
| "等数据1-2周" | 边等边建，每周5-8篇 | 良辰美案例 |
| 全A级打磨 | A20/B50/C30分级 | 同上 |
| Best列表KD 33-55太早 | KD<20优先，大词等6月 | Semrush实测 |
| 外链1%严重不足 | 每周固定1-2h，月+10-20 | 子木公式 |
| 技术SEO 29%过度 | 压缩到5%，够用就行 | 竞品分析 |
| 全站60+处"tested" | research-backed/analyzed | EEAT审计 |

### Reddit教训
- 旧号 Playwright自动化+Day 1放链接 → 10条全移除，账号已废
- 新号手动养号2周+D14解禁 → 存活，可安全放链接

### 竞品核心发现
- 6个中小宠物评测站 → 5个已死/停更 → **中层真空=机会窗口**
- pawsomeadvice: 806页/5年/723外链/DS 16 → 停更 → **无外链=死路**
- 良辰美: 300站/200K$/年，但黑帽手段已被2026年3月Spam Update打击
