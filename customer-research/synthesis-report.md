## 一、核心发现：两大平台的信号完美交叉验证

### 1.1 不锈钢饮水机 = 最大内容机会 ⭐⭐⭐⭐⭐

| 信号源 | 证据 |
|--------|------|
| **Google Trends** | 热度 23，远超所有品类词 |
| **Reddit** | "304 不锈钢"是绝对共识，chin acne 故事被反复引用 |
| **Amazon** | PETKIT "wireless pump" 设计被热捧，但"无线"一词引发大量 1 星差评 |
| **你的内容** | 已有 Best 列表 + 4 篇评测 + 2 对比，**缺一篇深度材质对比指南** |

→ **行动**：写"不锈钢 vs 陶瓷 vs 塑料猫饮水机终极指南（兽医怎么说）"

### 1.2 可靠性 > 一切 ⭐⭐⭐⭐⭐

| 平台 | 用户怎么说 |
|------|-----------|
| **Amazon** | "Kept disconnecting... it wouldn't even feed my cat" — 一票否决 |
| **Reddit** | "WiFi断了猫吃不到饭 = 灾难级故障" |
| **Amazon** | WOPET 3 个月卡粮 → 狗饿 4 天 → 信任崩塌 |
| **Reddit** | WOPET "App broken for months" 被反复鞭尸 |

→ **行动**：所有喂食器评测必须重点讨论"WiFi 断了怎么办？"有没有电池备份？

### 1.3 订阅费愤怒 = 跨品类转化杠杆 ⭐⭐⭐⭐

| Reddit | Amazon |
|--------|--------|
| Furbo $50 + $7/月 × 3 年 = $302 | PETLIBRO 订阅费"比喂食器本身还贵" |
| "选本地 SD 卡存储的型号" | 用户退货换机械式（零订阅） |
| 三大零订阅冠军：Eufy/Owlet/CINMOORE | — |

→ **行动**：每个有订阅费的产品页面加"3 年真实总成本"对比表

### 1.4 "Worth It" 是最高转化意图的关键词 ⭐⭐⭐⭐

Reddit 上占比最高的帖子类型就是"值不值得"。用户的决策逻辑是：

```
$700 / 36 个月 = $20/月 ≈ 每月少喝两杯咖啡
→ "有人每月帮我铲屎收 $20 我立马给"
```

→ **行动**：每篇评测加"$/月 算账"模块，把价格翻译成月成本
